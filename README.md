# HireScore Bias Investigation

## Overview

This repository contains the investigation of potential bias in **HireScore**, an AI-powered candidate ranking tool developed by TalentMatch AI. The investigation focuses on identifying, analyzing, and mitigating bias in the system, including gender, regional, and educational disparities in hiring outcomes.

The report includes:

- Bias type analysis with evidence  
- Bias pipeline mapping  
- Feature impact assessment  
- Mitigation strategies and action plans  

This work draws on industry best practices and research on AI fairness, bias mitigation techniques, and ethical AI considerations.

## Report

The full investigation report is available in:

[Bias Detective Investigation Report](bias_detective_investigation.md)


## Key Findings

- Overall bias severity: **High**  
- Primary affected groups: Female candidates, candidates from underrepresented regions, graduates of smaller universities  
- High-risk features: University attended, LinkedIn connections, previous employment location, extracurricular activities  
- Recommended actions: Feature adjustments, retraining with balanced data, human oversight, fairness monitoring
